# Règles spécifiques - Exemples

## Objectif
Fournir des exemples simples, réalistes et reproductibles.

## Principes
- Chaque exemple doit être autonome.
- Inclure un script de démarrage si nécessaire.
- Montrer uniquement les APIs publiques.

## Structure
- Exemples dans examples/.
- Un dossier par scénario.
