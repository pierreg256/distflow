# distflow

Framework TypeScript pour le développement distribué (PMD + messaging).

## Structure du projet

- packages/core/ : librairie principale
- packages/pmd/  : Port Mapper Daemon (PMD)
- packages/cli/  : CLI d’administration
- examples/      : exemples d’utilisation
- agents/        : règles et guides de développement
